## How It Works

**Browse** — Find your destination country using the search bar or explore the [interactive map](map.md).

**Verify** — Every data point links to its official source. Check the embassy portal yourself.

**Contribute** — Spot outdated info? Every page has an "Edit this page" button that takes you directly to the data source. No coding required.

## Built for Trust

- Every data point is backed by an official citation
- Community edits require source URLs — no unverified changes get merged
- Automated link health checks catch dead embassy URLs before you do
- Open source and transparent — [view the code](https://github.com/shan8tanu/atlas-wiki)
